Welcome to Arjun AI
Your personal AI agent for college life.
Uses Groq API — 100% Free, no credit card needed.
Step 1 — Get your free Groq key
1. Go to console.groq.com
2. Sign up free (Google login works)
3. Click API Keys → Create API Key
4. Copy the key (starts with gsk_...)
Step 2 — Paste your key here
🔒 Saved only in your browser — never sent anywhere except Groq directly.
▶ Is this really free? Is it safe?
Yes, completely free. Groq gives you free API access to Llama 3 — 30 requests/minute, 14,400 requests/day. No credit card ever needed.

Your key stays in your browser. It is saved using localStorage on your own computer. It is never uploaded to any server — your browser calls Groq directly.

To remove your key: Settings in the sidebar → API Key → 🗑 Remove.
⚠️ One small fix needed

You opened the app by double-clicking the file. Browsers block API calls when opened this way for security reasons.

Fix: open it through a local server instead. Takes 30 seconds:

Option A — Python (recommended)
Open terminal inside the arjun-ai folder, then run:
python3 -m http.server 3000
Then open http://localhost:3000 in your browser
Option B — Windows (double-click)
Double-click START.bat inside the arjun-ai folder.
It starts the server and opens the browser automatically.
Option C — VS Code (easiest for developers)
1. Install Live Server extension in VS Code
2. Right-click index.html → Open with Live Server
3. Done — auto-reloads when you edit files too
Or deploy to GitHub Pages (always works, no local setup) — see README.md
